# Iris & J Holdings Website

A modern, airy, premium website for Iris & J Holdings — a New Jersey real estate guidance and mobile notary brand led by Daiana Castro, REALTOR® and Mobile Notary.

This project is being designed to replace the current long, cluttered one-page website with a calm, organized, trust-building experience that helps visitors choose the right path quickly.

## Project Goal

The goal is not to build a flashy real estate template.

The goal is to build a simple, polished, human website that makes visitors feel:

- This person is organized.
- I know where to click.
- I understand what happens next.
- I can trust her with something important.

The website should feel like a calm concierge desk for real estate and notary services — light, spacious, warm, and professional.

## Design Rules

### Main Rule

The homepage should funnel users. It should not explain the entire business all at once.

Before adding anything to the homepage, ask:

> Does this help the visitor choose the right path?

If the answer is no, it belongs on a dedicated page, not the homepage.

### Cards

Cards should only be used when the visitor is choosing something.

Good uses of cards:

- Service selector on the homepage
- Resource options
- Admin dashboard summaries
- Lead type summaries

Avoid:

- Cards inside cards
- Every section trapped inside a card
- Heavy boxed layouts
- Random icon grids
- Overly busy sections

### Motion and Interaction

The site should feel interactive, but not chaotic.

Use:

- Soft card lift on hover
- Slight scale increase
- Gentle bounce return
- Smooth button hover states
- Subtle shadows
- Clean input focus states
- Soft page transitions

Avoid:

- Overdone animations
- Neon gradients
- Glassmorphism
- Floating blobs everywhere
- Startup-style motion overload

### Visual Style

Recommended direction:

- Warm white / cream background
- Deep navy or charcoal text
- Champagne / muted gold accents
- Soft tan neutrals
- Optional dusty blue or sage accent
- Elegant serif headings
- Clean sans-serif body text
- Large spacing
- Calm photography
- Minimal decorative elements

## Voice and Copy Rules

The wording should feel confident, warm, and human.

It should sound like a real person who is calm, prepared, and knows how to guide someone through a stressful process.

### Voice Should Be

- Clear
- Direct
- Warm
- Professional
- Calm
- Reassuring
- Confident without being pushy

### Avoid Robotic Phrases

Avoid wording like:

- Luxury-inspired client-focused brand
- Your trusted partner
- Seamless real estate journey
- Unlock your dream home
- Comprehensive real estate solutions
- Elevating your experience
- One-stop shop
- Bespoke solutions

### Preferred Style

Use plain, human language.

Example:

> Buying, selling, or signing important documents can feel like a lot. My job is to help you understand the next step, stay organized, and move forward with confidence.

### Public Site

- Mobile-first responsive layout
- Clean navigation
- Dedicated pages for each service
- Clear CTAs
- Fast loading images
- Accessible labels and alt text
- SEO metadata per page
- Legal footer on every page

### Forms

The site should support forms for:

- Buyer consultation
- Seller strategy call
- Home value request
- Mobile notary booking
- General contact
- Resource request

Recommended first version:

- Branded on-site flow
- External scheduling integration underneath, such as Google Calendar appointment scheduling, Calendly, or Cal.com

- Custom scheduling system
- Admin availability controls

- Email reminders
- SMS reminders

- Phone
- Email
- Service type
- Submitted details
- Appointment request if applicable

- Sell page image

## Current Decision

The homepage will stay simple.

It will not include every business feature.

The detailed service content, forms, vacation rentals, legal text, resources, and admin tools will live on their own pages.

This protects the premium feel and keeps the funnel clear.
